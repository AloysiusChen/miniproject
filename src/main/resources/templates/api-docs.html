API Documentation

Add Recommendation
Endpoint
POST /api/recommendations
Request Format

Content-Type: application/json

Example Request Body:
{
    "title": "Emergency Fund",
    "content": "Build 6 months of emergency savings",
    "ageMin": "18",
    "ageMax": "65",
    "dependents": "NONE",
    "links": [
        {
            "title": "Learn More",
            "url": "https://example.com/emergency-fund"
        }
    ]
}
Response Format

Success Response (200 OK):

{
    "message": "Recommendation added successfully",
    "id": "generated-unique-id"
}
Parameters
title (required)
The title of the recommendation.
content (required)
A detailed description or content for the recommendation.
ageMin (optional)
The minimum age for which the recommendation is relevant.
ageMax (optional)
The maximum age for which the recommendation is relevant.
dependents (optional)
The dependent type this recommendation is relevant for. Possible values:
  • "NONE" - For users with no dependents
  • "CHILDREN" - For users with children
  • "PARENTS" - For users with parents
  • "BOTH" - For users with both children and parents
links (optional)
A list of recommendation links, each containing:
  • title (string): The title of the link
  • url (string): The URL for the link
